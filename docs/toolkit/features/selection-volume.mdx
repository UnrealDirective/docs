---
title: "Selection Volume"
description: "A volume that can be placed and setup in the level to pre-define selecatable areas."
---

The Select Volume is a new actor that can be used to establish preset selection zones.

### Properties

| Name                    | Type          | Description                                                          |
| ----------------------- | ------------- | -------------------------------------------------------------------- |
| Box Extent              | Vector3D      | The adjustable size of the volume.                                   |
| Line Thickness          | Float         | The thickness of the volumes visible lines.                          |
| Line Color              | LinearColor2D | The color of the volumes lines.                                      |
| Select Contained Actors | Function      | Selects all actors that are completely within the bounds of the box. |
