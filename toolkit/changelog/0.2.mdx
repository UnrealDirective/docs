## New

- Added the Duplicate Asset Consolidator tool
- Added the `IsNotEmpty` text function to the Function Library

## Updated

- Toolkit deprecates support for Unreal Engine 5.0.3
  - _Note: This was done due to the code changes required between 5.0.3 and 5.1_
- Changed the `Enhanced Selection` tool name to `Actor Finder`

## Fixed

- Fixed the Actor Finder Mobility requiring a Class
- Fixed find actor crashes when attempting to iterate through invalid actors
