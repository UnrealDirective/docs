title: "Function Library"

The function library that comes with the Unreal Directive toolkit adds in quite a few helpful functions, with more function coming each update. These functions are exposed to both C++, Blueprints, and Python. The function library is broken up into categories, which are listed below.

---

## Editor Functions

Functions that enhance the Unreal Engine editor.

### GetChildClasses

```cpp
/**
 * Returns a list of actor classes that are derived from the given base class.
 * This exposes the built-in GetDerivedClasses function to blueprints.
 * @param BaseClass The base class to get the derived classes for.
 * @param bRecursive Whether to include derived classes of derived classes.
 * @param DerivedClasses The list of derived classes.
 */
```
