## Editor Functions

Functions that enhance the Unreal Engine editor.

### GetChildClasses

```cpp
/**
 * Returns a list of actor classes that are derived from the given base class.
 * This exposes the built-in GetDerivedClasses function to blueprints.
 * @param BaseClass The base class to get the derived classes for.
 * @param bRecursive Whether to include derived classes of derived classes.
 * @param DerivedClasses The list of derived classes.
 */
UFUNCTION(BlueprintPure, Category = "Unreal Directive Toolkit|Utility")
static void GetChildClasses(const UClass* BaseClass, bool bRecursive, TArray<UClass*>& DerivedClasses);
```

## Game Features

Functions that enhance Game Features inside of Unreal Engine.

### GetGameFeatureFromSubsystem

```cpp
/**
* Returns the Game Feature URL from the Game Feature Subsystem based on the given name.
* @param GameFeatureName - The name of the Game Feature.
*/
UFUNCTION(BlueprintPure, Category = "Unreal Directive Toolkit|Game Features")
static FString GetGameFeatureFromSubsystem(FString GameFeatureName);
```

### SetGameFeatureState

```cpp
/**
* Enable or disable a game feature.
* @param GameFeatureName - The name of the Game Feature.
* @param State - The state to set the Game Feature to.
*/
UFUNCTION(BlueprintCallable, Category = "Unreal Directive Toolkit|Game Features")
static void SetGameFeatureState(FString GameFeatureName, EUDGameFeatureState State);
```
